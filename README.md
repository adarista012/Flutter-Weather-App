# Flutter Weather App

A Weather App that is usgin https://openweathermap.org API

## How to run the flutter app

`inside lib/app/app_constants.dart file add an API_KEY`

`$ flutter pub get`

`$ flutter run`

## Usefull commands

`$ flutter clean`
